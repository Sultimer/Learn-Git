更新最后的提交
============

## 知识点

* git commit -m "commit message"
* git commit --amend
* git commit -am "commit message" [--amend]

## 实战演习

~~~bash
$ nano index.htm
...
$ git add .
$ git commit -m "modified."
$ git log
# remove debug info.
$ nano index.htm
...
$ git add .
$ git commit --amend
$ git status
$ git log
~~~
