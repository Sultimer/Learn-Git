Git初始设定
==========

## 知识点

* 建立一个Git库
* 设置基础信息
* 方便的命令

### 建立一个Git库

~~~bash
$ git init
~~~

### 设置基础信息

~~~bash
$ git init
$ git config -l
$ git config --global user.name "koma"
$ git config --global user.email "koma@komavideo.com"
$ git config --global color.ui true
$ git config -l
~~~

### 方便的命令

~~~bash
$ git config --help
$ git help config
~~~
