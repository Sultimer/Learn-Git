Git文件操作
==========

## 知识点

* git add [file1 file2 ...]
* git add .
* git rm
* git mv

## 实战演习

~~~bash
$ nano index.htm
...
$ nano style.css
...
$ git add .
$ git status
$ git mv index.htm index.html
$ git status
$ git rm --cached style.css
$ git status
~~~
