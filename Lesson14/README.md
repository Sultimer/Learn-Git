合并分支
=======

## 知识点

* git merge
* git branch -d [name]

## 实战演习

~~~bash
$ git branch
$ git checkout dev
$ nano index.htm
...
$ git add .
$ git commit -m "modified1."
$ git log
$ git branch
$ git checkout master
$ git log
$ git branch
$ git merge dev
$ git log
$ git branch -d dev
$ git branch
~~~
