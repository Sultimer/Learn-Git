把握Git状态
==========

## 知识点

* git status
* git checkout -- [file]

## 实战演习

~~~bash
$ nano index.htm
...
...
$ git status
$ git checkout -- index.htm
$ git status
$ nano index.htm
...
...
$ git add .
$ git reset HEAD index.htm
$ git checkout -- index.htm
$ git status
~~~
